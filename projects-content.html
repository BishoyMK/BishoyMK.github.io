Data Preprocessing & Power BI

Project Description

In this project, I conducted a comprehensive analysis of a company's employee dataset to uncover actionable business insights. The workflow involved data preprocessing using Python, machine learning evaluation, and interactive dashboard development with Power BI. Using Python, I cleaned and structured the raw dataset by handling missing values, removing duplicates, and normalizing data to ensure it was ready for meaningful analysis. I then applied machine learning models (such as classification algorithms) to predict employee retention based on features like department, salary, and years of experience, helping identify potential turnover risks.

To communicate insights effectively, I developed a Power BI dashboard featuring key visualizations, including:

  • Gender distribution
  • Annual salary distribution across departments
  • Employee count by department and country
  • Experience level segmentation
  • Geographic distribution on maps


The dashboard incorporates Power BI Bookmarks, enabling users to seamlessly toggle between multiple views and dive deeper into specific aspects of employee performance and structure. All preprocessing and model evaluation steps are documented in a Jupyter Notebook for reproducibility and clarity.

The final report was published through Power BI, allowing stakeholders to interact with data dynamically and make informed decisions based on clear metrics and trends.

Technologies Used

  • • Python (Pandas, NumPy)
  • • Power BI
  • • Data Visualization Tools

Features

  • • Data Cleaning and Preprocessing
  • • Interactive Dashboards
  • • Employee Performance Analytics
  • • Custom Visualizations

GenAASL–Text-to-Avatar American Sign Language Video Generation

Project Description

For my graduation project at Egypt-Japan University of Science and Technology (E-JUST), my team and I developed GenAASL, a cutting-edge system designed to bridge the communication gap between the hearing and Deaf communities. The project converts English text into American Sign Language (ASL) videos using both real signer footage and a 3D animated avatar. We tackled the linguistic challenge of aligning English with ASL grammar by leveraging large language models (LLMs)—specifically Claude 3-7 Sonnet—for accurate gloss generation.

A key innovation in the project was the avatar-based sign generation module. To overcome inconsistencies in signer appearance and signing style found in real video datasets, we used the ExPose model—a high-resolution 3D human mesh framework fine-tuned on the ASLLVD dataset. This allowed us to generate realistic, expressive, and consistent avatar animations that captured critical ASL features such as handshapes, facial expressions, and upper-body motion. The resulting avatar videos offer a scalable, high-quality alternative to human signers, especially useful in on-demand translation contexts.

Hosted on AWS cloud infrastructure, GenAASL integrates components like Lambda functions, S3 storage, and DynamoDB to ensure fast, modular, and real-time performance. The project showcases the power of AI, motion modeling, and cloud computing to deliver inclusive, accessible technology that serves the Deaf community in educational, digital, and public settings.

Technologies Used

  • • Large Language Models (LLMs): Claude 3-7 Sonnet via AWS Bedrock (for ASL gloss generation)
  • • 3D Animation & Pose Estimation: ExPose model (fine-tuned for sign language motion)
  • • Cloud Services: AWS (Lambda for gloss generation, S3 for dataset and video storage, DynamoDB for metadata management)
  • • Programming Languages: Python, JavaScript
  • • Web Interface: HTML, CSS, Flask (or similar) for input/output interaction
  • • Dataset: ASLLVD (American Sign Language Lexicon Video Dataset)

Features

  • • English-to-ASL Translation: Converts English text into grammatically correct ASL gloss using large language models.
  • • 3D Avatar Sign Generation: Produces realistic ASL videos with detailed hand, face, and body motion using the ExPose model.
  • • Cloud-Based System: Deployed on AWS for scalable, real-time performance with efficient data handling and storage.

Holy Family - Day Nursery School Website

Project Description

The Holy Family Day Nursery School Website is a responsive and user-friendly web application developed using React. The site features multiple structured pages including Home, About Us, Program, Admissions, Parents, Homework, and Contact Us. Each page was designed with clarity, accessibility, and performance in mind, offering an intuitive interface for parents to explore the school's offerings, values, and updates. The website is visually appealing and reflects the welcoming spirit of the nursery, with clean layouts and a calm, professional color scheme.

I was responsible for developing the entire frontend, implementing mobile responsiveness, navigation, and form functionality using modern web technologies. I ensured consistent design across devices and screen sizes, using responsive design principles and components that adapt fluidly. The Contact Us page includes a styled interactive form, and all navigation links are fully functional. This project demonstrates my skills in frontend development, user interface design, and building accessible web experiences that meet real-world needs.
